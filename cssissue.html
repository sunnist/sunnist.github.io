全局修改

字体为

font-family: "Hiragino Sans GB","冬青黑体","Microsoft YaHei","微软雅黑","Arial";

普通正文大小为16px，颜色为 #212121，高亮颜色为#883a36

首页列表修改

1、左边封面图去掉 hover 阴影

2、右边料理活动的时间、价格、预订人员文字需要高亮

.home-feast-front-meta, .home-feast-front-reserTip {
margin-bottom: 5px;
font-size: 14px;
color: #883a36;
}

首页展开详细修改

1、料理活动的介绍文字大小为16px，间距为2px ，这里 width 最好设置为760px，与下面最宽的图片展示区域对齐

  .home-feast-detail-host p {
   font-size: 16px;
   letter-spacing: 2px;
   }

2、去掉料理人头像的阴影，减少上下 margin 值 

.home-feast-detail-host .avatar {
 width: 96px;
 height: 96px;  
 margin-top: 30px;
 margin-bottom: 30px;
 /* -webkit-box-shadow: 0 0 10px rgba(0,0,0,.5); */
/* box-shadow: 0 0 10px rgba(0,0,0,.1); */
}

3、 减少料理人名的 margin-bottom 值

 .home-feast-detail-host h2 {
  margin-bottom: 15px;
 font-weight: 500;
}

4、减少分割线的上下 margin 值

.home-feast-detail .divider {
 margin-top: 30px;
 margin-bottom: 30px;
}

5、目前料理人、每个模块的标题是 h2，希望视觉效果是18px（标题加粗），至于这个h2设置为 18px 可能是不符合规范的

  h2 {
  font-size: 18px;
  margin-top: 15px;
  margin-bottom: 15px;
 }

6、减少每块内容标题的 margin-bottom 

.home-feast-detail-sub-section .home-feast-detail-sub-title {
  margin-bottom: 15px;
  line-height: 1.2;
}

7、减少每块内容主题的 margin-bottom

.home-feast-detail-sub-section {
  margin-bottom: 30px;
}

8、菜单字体大小为16px，这里后面的特色文字也是16px

.home-feast-detail-menu-item {
  margin-bottom: 18px;
  font-size: 16px;
}

9、去掉特色图片的阴影

.home-feast-detail-char-images .item {
  display: block;
  float: left;
 width: 350px;
  height: 350px;
  margin: 15px;
  background-position: 50% 50%;
  background-size: cover;
  /* -webkit-box-shadow: 0 0 10px #000; */
  /* box-shadow: 0 0 10px #000; */
}
